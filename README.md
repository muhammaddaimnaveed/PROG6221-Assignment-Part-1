Cybersecurity Awareness Chatbot

This project is a simple Cybersecurity Awareness Chatbot that is designed to help users understand basic cybersecurity concepts
and how to protect themselves from cyber threats online. When the user runs the program, the program will display the Chatbots 
ASCII art logo titled “Cyber Secure”, followed by a greeting audio, and a welcome message. The user will then be required to 
enter their name, and with this, they will get a personalise response using their name to get a more engaging and interactive 
user experience. After the introduction, the chatbot presents the users with a menu of predefined questions that relate to 
cybersecurity, allowing them to select an option and enter the query to get a response from the Chatbot. These responses are 
designed to educate users on a variety of topics such as phishing, password safety, recognising suspicious activity, and staying 
safe while browsing the internet. Overall, the chatbot serves as an interactive educational tool that is designed to educate 
users about important cybersecurity concepts and how to protect themselves and stay safe online.

System Setup for Beginners:

1. Install and open Visual Studio
2. Make sure you have .Net Framework installed
3. Download or clone your repository
4. Open the project in Visual Studio
5. Click build to build the solution
6. Click start to run the program

Usage Intructions:

1. How are you?
2. What is your purpose?
3. What can I ask you?
4. What is password safety?
5. What is phishing?
6. How can I practice safe browsing? 
7. How do I create a strong password?
8. What is two-factor authentication? 
9. How do I recognise a secure website?
10. How can I protect my personal information?
11. Exit

Current Features:

1. Voice greeting when the program starts
2. ASCII art display
3. Personalised user interaction 
4. Basic chatbot responses
5. Input validation for incorrect entries
6. Colourful console interface
